# 📊 Sales Performance Dashboard - Power BI Project

## 🚀 Overview
This project is an interactive Sales Performance Dashboard built using Microsoft Power BI.

The dashboard helps analyze:
- Sales Performance
- Profit Trends
- Customer Insights
- Payment Mode Analysis
- Category & Sub-Category Performance

It provides business insights through interactive charts, KPI cards, and filters.

---

## 📌 Dashboard Features

✅ KPI Cards  
- Total Sales  
- Total Profit  
- Total Quantity  
- Total Orders  

✅ Interactive Filters  
- Quarter Filter  
- Category Filter  
- Payment Mode Filter  

✅ Visualizations  
- Monthly Profit Trend  
- Profit by Category  
- Sub-Category Analysis  
- Customer Insights  
- Payment Mode Distribution  

---

## 🛠 Tools & Technologies Used

- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Data Visualization
- Dashboard Design
- Data Cleaning & Transformation
